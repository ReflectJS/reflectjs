Reactive HTML

In Reflect.js pages you can add logic values to any tag and use responsive JavaScript expressions in any attribute or text to reflect results in the page. 

<div :count="[['[[']]0]]"
     :did-init="[['[[']]
       setInterval(() => count++, 1000)
     ]]">
  Seconds: [['[[']]count]]
</div>
Seconds: [[count]]
